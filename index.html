Our Guidr Project

Our team was tasked to create an app that users can up as a guide and sign in as a guide with username and credentials. Once signed up the authenticated guide can access a web or mobile view. In both views users should be able to see a list of previous trips they have created, select a trip from the list and be presented with a detailed view that displays all the trip's properties, create a new trip, update a trip and delete a trip.
